Privacy Policy

Temporary Download Cleaner is a Chrome extension that automatically deletes downloaded files after a user-defined timer.

Data Collection

This extension does not collect, store, or transmit personal user data.

The extension does not collect:

  • Name
  • Email address
  • Browsing history
  • Location data
  • Account information

Local Storage

User preferences are stored locally using the Chrome chrome.storage.local API.

This includes:

  • Timer presets
  • User configuration settings
  • Active timers for temporary downloads

All data remains on the user's device and is never transmitted externally.

Downloads Access

The extension uses the Chrome downloads API only to detect completed downloads and optionally delete files after a timer expires.

Notifications

Browser notifications may be used to inform the user when a temporary download has been automatically deleted.

Third-Party Services

The extension does not use:

  • Analytics services
  • Advertising networks
  • Tracking technologies
  • Remote code

Changes

This policy may be updated if the extension functionality changes.

# Temporary Download Cleaner

Temporary Download Cleaner is a Chrome extension that automatically deletes downloaded files after a configurable timer.

The extension helps users manage temporary downloads such as installers, documents, and media files that are no longer needed.

## Features

- Automatically delete downloaded files after a timer
- Configure expiration time
- Keep files permanently if needed
- Lightweight and privacy-friendly

## Privacy

This extension does **not collect, store, or transmit personal user data**.

All configuration data is stored locally using the Chrome Storage API.

Privacy policy:

https://ezghost1.github.io/temporary-download-cleaner/privacy/

## Permissions

The extension uses the following Chrome APIs strictly for its functionality:

- downloads
- downloads.open
- storage
- alarms
- notifications
- tabs
- scripting
